# Repository Scafolding

The repo structure is as follows:

* weekX folder contains all the solutions to week tasks

* weekX/taskX contains the specific task solution

* weekX/taskX/README.md contains information about the task, how to run it, and architecture decisions

* weekX/README.md covers some of the lessons learned during the week
