# Task Summary

Geospatial service for storing and finding locations.

Full specification:

> https://github.com/HackBulgaria/NodeJS-1/tree/master/week2/3-Geolocation-and-stuff


# How to Run the Solution

* run mongodb instance on your localhost

* do ```npm i``` to install dependencies for the server app

* run the server with ```npm start``` (```npm run start-dev``` for development mode)

> see config.json for details about mongodb settings used in express server

* do ```bower install``` in both client apps

> save-it for saving locations

> find-it for finding locations

* open save-it/index.html and find-it/index.html in your browser

> use save it to store locations

> use find it to find locations
